# Customer-Behavior-Analysis

*	Project Summary
This project analyzes customer shopping behavior data to understand purchasing patterns, revenue trends, and customer segments.
I performed end-to-end data analysis using Python and MySQL, starting from data cleaning to generating business insights through SQL queries.

*	Tools Used
Python (Pandas)
MySQL
SQLAlchemy
SQL (CTE, Window Functions, Aggregations)

* Project Workflow
Cleaned and preprocessed raw CSV data
Handled missing values using category-wise median
Created new features like age groups and purchase frequency (in days)
Uploaded cleaned data to MySQL database
Wrote SQL queries to answer business questions

*	Key Analysis Performed
Revenue comparison by gender
Impact of subscription on spending
Top 5 highest-rated products
Products with highest discount usage
Customer segmentation (New, Returning, Loyal)
Revenue contribution by age group

Data Cleaning & Feature Engineering

Business Insight Generation
